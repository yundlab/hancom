이 페이지 코드 안내

모든 예제는 ES6+(2015년 이후 최신 자바스크립트) 문법으로 작성
• 변수는 let·상수는 const (옛날 var 안 씀)
• 함수는 화살표 함수 ( ) => { } 사용
• 문장 조립은 템플릿 리터럴 `${ }` (옛날 + 이어붙이기 대신)
• 색깔 규칙 — 상수·변수·메서드·함수·속성을 색으로 구분
