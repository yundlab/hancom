# HuggingFace 토큰 발급·설정

## 발급
1. https://huggingface.co/settings/tokens 접속
2. **Create new token** → 타입 **Read**(classic) 또는 fine-grained + "Make calls to Inference Providers" 체크
3. 토큰 복사 (한 번만 전체 표시됨)

## 설정 (.env — 따옴표 없이)
```
HF_TOKEN=hf_발급받은토큰
```

## 코드에서 사용
```python
import os
from pathlib import Path
from dotenv import load_dotenv
load_dotenv(Path(__file__).with_name(".env"))
token = os.environ["HF_TOKEN"]
```

⚠️ 토큰은 절대 코드·주석·마크다운에 직접 쓰지 말 것. `.env`(gitignore됨)에만 저장.
