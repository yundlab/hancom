# 개발자도구 (F12) — 화면 뒤를 들여다보는 꿀팁

브라우저에 기본 내장된 도구로, 화면 뒤 HTML·CSS를 바로 확인하고 값을 잠깐 바꿔 미리볼 수 있어요.  
에러 잡기, 디자인 확인 등에 쓰이며 별도 설치 없이 크롬·엣지에서 바로 사용 가능해요.

---

## 여는 법

| 방법 | 단축키 |
|---|---|
| 키보드 | `F12` (Mac: `Cmd + Opt + I`) |
| 마우스 | 화면 아무 곳 우클릭 → **검사** |

---

## 주요 기능

### 1. 요소 검사 (화살표 아이콘)
화살표 아이콘 클릭 후 화면 요소에 마우스를 올리면  
**그 부분이 무슨 태그인지** 콕 짚어줘요.

### 2. Elements 탭 — HTML 구조 보기
페이지의 전체 HTML 구조를 트리 형태로 보여줘요.  
줄을 클릭하면 화면에서 **해당 부분이 반짝** 표시돼요.

### 3. 값 직접 수정해보기
글자·색상 값을 **더블클릭해서 바꾸면** 화면에 바로 반영돼요.  
디자인을 미리 확인할 때 유용해요.

### 4. Console 탭 — 에러 확인
빨간 글씨로 표시된 에러 메시지를 보고  
**어디가 잘못됐는지** 힌트를 얻을 수 있어요.

---

## 주의사항

> 개발자도구에서 바꾼 내용은 **화면에서만 잠깐** 적용돼요.  
> 내 파일은 변경되지 않고, 새로고침하면 원래대로 돌아와요.  
> 마음껏 실험해도 안전해요.

---

## 꿀팁

잘 만들어진 사이트에서 `F12`를 열어 구조를 구경하는 게  
**가장 빠른 HTML·CSS 공부법**이에요.
